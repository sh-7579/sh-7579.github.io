個人サイト用のレポジトリです．

🔗 https://sh-7579.github.io

## 概要

- [al-folio](https://github.com/alshedivat/al-folio) テーマ（Jekyll）をベースに構築

## ディレクトリ構成

```
_pages/       # 各ページのコンテンツ（about, cv など）
_posts/       # ブログ記事
_projects/    # プロジェクト一覧
_bibliography/ # 論文リスト（BibTeX）
assets/       # 画像・CSS・JS
_config.yml   # サイト全体の設定
```
